Python

Maestria no uso da linguagem, tratamento de strings, automação, expressões regulares, escrita limpa e conhecimento de uma vasta gama de frameworks.

Pandas

Mineração e tratamento de dados, séries temporais, capacidade de explorar e identificar relacionamento e correlação entre os dados.

Uso da ferramenta para ler, escrever e manipular arquivos Excel(.xlsx), automações de diversas tarefas relacionadas às planilhas, semelhante ao VBA

Uso da lib para vetorizar operações matemáticas e estatísticas, consumindo menos recurso, e auxiliando no tratamento dos dados.

Conhecimento de uma variedade de algoritmos de machine learn e técnicas de modelagem de dados para explorar e analisar dados de maneira mais profunda e precisa.

Automação para navegadores web, permitindo realizar tarefas repetitivas de forma programática, como coleta de dados por exemplo.
